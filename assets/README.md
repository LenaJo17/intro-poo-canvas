# 🎨 Canvas 2D App

Aplicación web desarrollada con **HTML, CSS, Bootstrap y JavaScript** que demuestra el uso de la **Canvas API** utilizando **Programación Orientada a Objetos (OOP)** para dibujar objetos gráficos en pantalla.

Este proyecto muestra cómo crear, manipular y renderizar **objetos 2D** dentro de un elemento `<canvas>`.

---

# 📷 Vista de la aplicación

La aplicación contiene tres ejemplos de objetos dibujados con Canvas:

### 1️⃣ Objeto 2D

Muestra un círculo generado mediante una clase utilizando **programación orientada a objetos**.

### 2️⃣ Objeto Random

Genera un círculo con **posición y tamaño aleatorios** dentro del canvas.

### 3️⃣ Múltiples Objetos

Genera **varios círculos con diferentes tamaños y colores** dentro del canvas sin salirse de los límites.

# ⚙️ Funcionamiento del proyecto

El proyecto utiliza una **clase llamada `Circle`** que define las propiedades de un objeto gráfico.

### Propiedades principales

* `x` → posición horizontal
* `y` → posición vertical
* `radius` → tamaño del círculo
* `color` → color del borde
* `text` → texto dentro del círculo
* `backcolor` → color de relleno
* `textColor` → color del texto

### Método principal

```
draw(context)
```

Este método permite **dibujar el objeto dentro del canvas** usando el contexto 2D.
